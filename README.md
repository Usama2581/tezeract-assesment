<h2>Backend is deployed on cyclic</h2>

<h3>To get all employees</h3>
https://excited-frock-fawn.cyclic.app/employee/get

<h3>To get the average salary of an employee by position e.g Frontend Developer & Backend Developer</h3>
https://excited-frock-fawn.cyclic.app/employee/average-salary/Backend Developer
https://excited-frock-fawn.cyclic.app/employee/average-salary/Frontend Developer

<h3>To filter employees by experience level</h3>
https://excited-frock-fawn.cyclic.app/employee/experience?lessThan=2&greaterThan=3

<h3>To get the top earner. The URL below returns 5 top earners </h3>
https://excited-frock-fawn.cyclic.app/employee/top-earner/5

<h3>Check the retention rate</h3>
https://excited-frock-fawn.cyclic.app/employee/retention-rate?employeeAtStart=50&employeeAtEnd=40&employeeWhoLeft=10

<h3>To get the employees of given salary range</h3>
https://excited-frock-fawn.cyclic.app/employee/salary-range?greaterThan=100000&lessThan=1500000
